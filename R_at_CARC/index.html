Runing R at CARC intro

The following markdown files contain the introductory workshop material for running R on CARC systems.

The "tutorials" go basically in this order:

  1. Getting R software
  2. Installing packages
  3. PBS job submission

  4. A GNU parallel tutorial

    The following files demonstrate parallellization of R code on a CARC system: * sequential_r.pbs is a job submission script to submit the super simple sequential.R job * parallel_r.pbs is a job submission script to submit the parallel.r * parameters
